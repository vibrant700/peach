# 智桃科研 - 作品安装说明

## 作品概述

本作品为智慧农业综合服务平台，采用端云协同架构，包含Android移动应用、Node.js后端服务和Python AI模型引擎。支持桃子品种识别、品质检测、智能咨询、社区交流、文创设计、库存管理、碳汇交易等功能。

## 系统要求

### 硬件要求

**最低配置：**
- 处理器：Intel Core i5及以上或同等性能处理器
- 内存：8GB RAM
- 存储：20GB可用空间
- 网络：稳定的网络连接

**推荐配置：**
- 处理器：Intel Core i7及以上
- 内存：16GB RAM
- 存储：50GB可用空间（包含AI模型文件）
- 网络：有线网络连接，带宽≥100Mbps

**移动设备要求：**
- Android 7.0（API Level 24）及以上
- RAM ≥ 2GB
- 存储空间 ≥ 500MB
- 摄像头功能（用于拍照识别）
- 网络连接（WiFi或移动数据）

### 软件要求

**开发环境（如需二次开发）：**
- Android Studio 2022.1.1及以上
- Node.js 16.x及以上
- Python 3.8及以上
- Git版本控制工具

**运行环境（部署使用）：**
- Windows 10/11或macOS 11及以上或Linux（Ubuntu 20.04 LTS）
- Android 7.0及以上移动设备

## 安装前准备

### 1. 环境检查

在开始安装前，请确认系统满足以下条件：

- 已安装Java Development Kit (JDK) 8或更高版本
- 已安装Node.js运行环境
- 已安装Python 3.8或更高版本
- 确保有管理员权限（用于安装软件和配置环境）

### 2. 获取安装包

本作品提供两种安装方式：

**方式一：完整安装包（推荐）**

- 文件名：智桃科研_完整安装包_v1.0.zip

**方式二：源代码安装**
- 从GitHub仓库克隆源代码

## 安装步骤

### 步骤一：后端服务安装

#### 1.1 解压安装包

将下载的完整安装包解压到目标目录，建议路径：
- Windows: `D:\PeachResearch\`
- macOS: `/Applications/PeachResearch/`
- Linux: `/opt/peach-research/`

#### 1.2 Node.js服务配置

进入解压后的目录，找到`android`文件夹：

```bash
cd android
```

#### 1.3 安装依赖

在命令行中执行以下命令：

```bash
npm install
```

等待依赖安装完成，这可能需要几分钟时间。

#### 1.4 Python环境配置

确保Python 3.8+已安装，然后配置虚拟环境：

```bash
cd yolov5-master
python -m venv .venv
```

激活虚拟环境：

- Windows: `.venv\Scripts\activate`
- macOS/Linux: `source .venv/bin/activate`

安装Python依赖：

```bash
pip install -r requirements.txt
```

#### 1.5 AI模型文件放置

将预训练模型文件`best.pt`放置到`yolov5-master`目录下。模型文件通常包含在完整安装包中。

#### 1.6 启动后端服务

在`android`目录下执行：

```bash
node shibie.js
```

看到以下信息表示启动成功：

```
==========================================
  桃子识别服务已启动
  本机IP: 192.168.1.100
  访问地址为 http://192.168.1.100:8081
  上传接口: http://192.168.1.100:8081/upload
==========================================
```

### 步骤二：Android应用安装

#### 2.1 安装APK文件

在移动设备上打开设置，允许安装来自未知来源的应用。

通过以下任一方式安装：

**方式一：直接安装**
- 将`app-release.apk`文件传输到手机
- 在文件管理器中点击APK文件
- 按照提示完成安装

**方式二：通过Android Studio**
- 连接手机到电脑，开启USB调试
- 在Android Studio中打开项目
- 点击Run按钮或使用快捷键部署

#### 2.2 配置服务器地址

首次启动应用时，需要配置服务器地址：

1. 打开"慧眼识桃"功能
2. 应用会自动检测局域网内的服务器
3. 如检测失败，手动输入服务器IP地址（如：192.168.1.100:8081）
4. 点击测试连接，确保通信正常

#### 2.3 AI服务配置（可选）

如需使用"淘宝对话"功能，需要配置AI服务：

1. 获取DeepSeek API Key（访问 https://platform.deepseek.com/）
2. 打开`app/build.gradle`文件
3. 找到`DEEPSEEK_API_KEY`配置项
4. 填入您的API Key
5. 重新编译安装应用

或者使用本地部署的Ollama服务，无需API Key。

### 步骤三：功能测试

#### 3.1 品种识别测试

1. 打开应用，点击"慧眼识桃"
2. 点击"上传图片"或拍照
3. 选择一张桃子图片
4. 点击"开始识别"
5. 等待3-5秒，查看识别结果

#### 3.2 智能对话测试

1. 返回主界面，点击"淘宝对话"
2. 在输入框中输入问题，如"桃子如何保鲜？"
3. 点击发送，等待AI回复
4. 尝试切换不同模式（通用、农业、推荐）

#### 3.3 其他功能测试

依次测试以下功能：
- 桃农社区：发布帖子、评论互动
- 桃源工坊：生成文创设计图片
- 仓库管理：添加库存、搜索物品
- 碳汇市场：模拟交易、查看行情

## 配置说明

### 网络配置

#### 局域网部署（推荐）

**适用场景：** 家庭、实验室、办公室等局域网环境

**配置步骤：**
1. 确保服务器和手机连接同一WiFi
2. 在服务器上运行`ipconfig`（Windows）或`ifconfig`（macOS/Linux）
3. 查看本机IP地址，如：192.168.1.100
4. 在Android应用中输入该IP地址和端口8081

**优势：** 延迟低、稳定性好、不消耗流量

#### 公网部署

**适用场景：** 远程访问、多地点使用

**配置步骤：**
1. 配置路由器端口转发（外部端口→内部8081端口）
2. 申请域名并配置DNS解析
3. 配置SSL证书（推荐使用Let's Encrypt免费证书）
4. 在Android应用中输入公网地址和域名

**注意事项：**
- 确保服务器安全性，设置强密码
- 定期更新系统和服务
- 监控服务器运行状态

### AI模型配置

#### YOLOv11模型

模型文件位置：`yolov5-master/best.pt`

**自定义训练：**
1. 准备桃子数据集（VOC或YOLO格式）
2. 修改`yolov5-master/data/`下的配置文件
3. 运行训练命令：`python train.py --data custom.yaml --weights yolov5s.pt`
4. 训练完成后，将生成的`best.pt`替换原模型文件

#### 甜度检测模型

光谱分析模型基于7个特征波长点，无需额外配置。

如需校准：
1. 使用标准样本采集光谱数据
2. 使用Excel或Python重新拟合线性回归模型
3. 更新应用中的预测参数

#### 着色度检测算法

HSV算法参数已在应用中预设，支持自动光照适应。

如需调整阈值：
1. 打开`IdentifyPeachActivity.java`
2. 找到HSV阈值配置部分
3. 根据实际环境调整参数范围

## 常见问题排查

### 问题1：后端服务无法启动

**症状：** 运行`node shibie.js`后报错或立即退出

**可能原因及解决方案：**

1. **端口被占用**
   - 检查8081端口是否被其他程序占用
   - 修改`shibie.js`中的端口号
   - 或关闭占用端口的程序

2. **依赖安装不完整**
   - 删除`node_modules`文件夹
   - 重新运行`npm install`
   - 确保网络连接正常

3. **Python环境问题**
   - 确认Python版本（3.8+）
   - 检查虚拟环境是否正确激活
   - 重新安装Python依赖

### 问题2：Android应用无法连接服务器

**症状：** 上传图片时提示网络错误或连接超时

**可能原因及解决方案：**

1. **网络不在同一局域网**
   - 确保手机和服务器连接同一WiFi
   - 检查路由器是否开启了AP隔离
   - 尝试使用有线网络连接服务器

2. **防火墙阻止**
   - 临时关闭防火墙测试
   - 或添加防火墙规则允许8081端口
   - Windows防火墙设置：控制面板→系统和安全→Windows Defender防火墙→高级设置

3. **IP地址错误**
   - 重新确认服务器IP地址
   - 使用`ping`命令测试连通性
   - 在应用中重新输入服务器地址

### 问题3：识别结果不准确

**症状：** 识别错误、置信度低或无法识别

**可能原因及解决方案：**

1. **图片质量问题**
   - 确保图片清晰，光线充足
   - 避免模糊、过暗或过曝的图片
   - 建议使用高分辨率图片

2. **模型版本问题**
   - 确认使用的是最新训练的模型文件
   - 检查模型文件是否完整（约14MB）
   - 重新下载或训练模型

3. **背景复杂**
   - 尽量选择单一背景的图片
   - 避免枝叶、果实重叠
   - 使用手动裁剪功能聚焦目标

### 问题4：AI对话功能异常

**症状：** 对话无响应、报错或回复异常

**可能原因及解决方案：**

1. **API Key配置错误**
   - 检查API Key是否正确填入
   - 确认API Key未过期
   - 检查账户余额是否充足

2. **网络连接问题**
   - 确认设备能访问外网
   - 检查DNS设置是否正确
   - 尝试切换网络（WiFi↔移动数据）

3. **本地Ollama配置**
   - 如使用本地部署，确认Ollama服务运行正常
   - 检查模型是否已下载
   - 查看Ollama日志排查问题

### 问题5：应用闪退或卡顿

**症状：** 应用意外关闭或运行缓慢

**可能原因及解决方案：**

1. **内存不足**
   - 关闭其他后台应用
   - 清理应用缓存
   - 重启手机释放内存

2. **版本兼容性**
   - 确认Android版本≥7.0
   - 更新应用到最新版本
   - 检查设备架构是否支持（armeabi-v7a、arm64-v8a）

3. **存储空间不足**
   - 清理手机存储空间
   - 卸载不常用的应用
   - 确保至少有500MB可用空间

## 卸载说明

### Android应用卸载

1. 在手机设置中找到"应用管理"
2. 找到"智桃科研"应用
3. 点击"卸载"按钮
4. 确认卸载操作

**注意：** 卸载应用会清除所有本地数据，包括聊天记录、库存信息等。

### 后端服务卸载

#### Windows系统

1. 停止Node.js服务（按Ctrl+C）
2. 删除安装文件夹
3. 清理环境变量（如已配置）

#### macOS/Linux系统

1. 停止Node.js服务（按Ctrl+C）
2. 删除安装文件夹
3. 清理软链接（如已创建）

**完全清理命令（Linux）：**

```bash
# 停止服务
pkill -f "node shibie.js"

# 删除安装目录
rm -rf /opt/peach-research

# 清理npm缓存（可选）
npm cache clean --force
```

**方式一：覆盖安装**

2. 备份配置文件和模型文件
3. 下载最新版安装包
4. 解压并替换旧文件

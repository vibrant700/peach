# 智桃科研 - 六大核心功能介绍

## 🍑 1. 慧眼识桃

**功能定位**：AI智能品种识别与品质检测

**核心功能**：
- **品种识别**：基于YOLOv5深度学习模型，精准识别桃子品种（北京27号、春蜜、白风等）
- **品质评估**：综合分析甜度、成熟度、果实尺寸、光照情况等多项指标
- **加工建议**：根据检测结果智能推荐最佳加工方式（桃干、果脯、桃酒、桃汁、罐头、果酱等）
- **可视化报告**：生成详细的检测报告，包括匹配度评分和加工优势分析

**技术栈**：
- **前端**：Android原生应用
- **AI模型**：YOLOv5目标检测 + PyTorch
- **后端**：Node.js + Express + Multer
- **通信**：HTTP + multipart/form-data
- **图像处理**：Bitmap + Base64编码

**应用场景**：
- 果农快速了解桃子品质和最佳利用方式
- 加工企业原材料分级和采购决策
- 科研机构品种鉴定和数据收集

## 🤖 2. 桃宝对话

**功能定位**：智能农业助手与专家咨询系统

**核心功能**：
- **通用咨询**：日常生活问题解答，提供各类信息查询
- **农业问答**：桃树栽培技术、病虫害防治、施肥灌溉专业知识
- **导购建议**：桃产品购买指导、品种对比、性价比分析
- **智能对话**：基于DeepSeek-R1大模型，上下文理解能力强

**技术栈**：
- **AI模型**：DeepSeek-R1 (67B参数)
- **推理引擎**：Ollama本地部署
- **通信**：HTTP API + 流式输出
- **模式**：System Prompt角色切换
- **网络**：10.0.2.2本地转发

**适用人群**：
- 种植户获取专业农技指导
- 消费者选购桃产品咨询
- 科研人员知识查询辅助

## 🌾 3. 桃农社区

**功能定位**：种植经验交流与知识分享平台

**核心功能**：
- **经验分享**：发布种植心得、栽培技巧、病虫害防治经验
- **技术讨论**：专家在线答疑、农户互助交流
- **市场信息**：价格行情、供需信息、产销对接
- **成功案例**：优秀种植案例展示与学习

**技术栈**：
- **前端**：Android RecyclerView + CardView
- **UI设计**：Material Design组件
- **交互**：评论/点赞/分享功能
- **数据**：本地SQLite存储
- **扩展**：预留RESTful API接口

**价值体现**：
- 新手农户快速学习
- 老手农户经验变现
- 产销对接减少中间环节
- 共建产业知识库

## 🎨 4. 桃源工坊

**功能定位**：文创设计与个性化定制平台

**核心功能**：
- **Logo设计**：桃品牌标识、包装图案自动生成
- **海报制作**：产品宣传海报、活动展板智能设计
- **IP形象**：桃子卡通形象、品牌吉祥物创作
- **包装设计**：产品包装盒、礼盒、标签设计
- **文创周边**：帆布袋、徽章、明信片等衍生品设计

**技术栈**：
- **AI生成**：阿里云通义千问（DashScope API）
- **图像模型**：文生图模型（wanx-v1）
- **API通信**：HTTPS + JSON
- **图像加载**：Glide图片加载库
- **密钥管理**：BuildConfig配置API Key

**应用方向**：
- 桃产品品牌化升级
- 农文旅融合发展
- 特色伴手礼设计
- 企业形象建设

## 📦 5. 仓库管理

**功能定位**：农产品智能仓储与库存管理

**核心功能**：
- **入库管理**：批次录入、质量检验、库位分配
- **库存监控**：实时库存、保质期提醒、温湿度监控
- **出库管理**：订单拣货、批次追溯、损耗统计
- **数据分析**：周转率分析、库存预警、采购建议
- **可视报表**：库存结构图表、出入库趋势分析

**技术栈**：
- **数据库**：Room Persistence Library (SQLite)
- **架构**：MVVM + Repository模式
- **扫描**：CameraX + ML Kit barcode扫描
- **图表**：MPAndroidChart可视化
- **通知**：WorkManager定时任务

**核心价值**：
- 减少库存损耗
- 提高周转效率
- 保障产品质量
- 降低管理成本

## 🌍 6. 碳汇市场

**功能定位**：农业碳汇交易与碳中和管理

**核心功能**：
- **碳汇计算**：桃园固碳量科学测算
- **碳汇交易**：碳信用买卖、供需对接
- **碳中和认证**：碳足迹评估、碳标签申请
- **数据分析**：碳汇趋势、价格行情、政策解读
- **项目申报**：碳汇项目备案、核查辅导

**技术栈**：
- **测算模型**：果园固碳量计算公式
- **数据来源**：遥感数据 + 地面监测
- **交易**：模拟交易撮合引擎
- **存证**：区块链哈希存证（预留）
- **展示**：实时行情图表

**市场价值**：
- 增加农户额外收入
- 助力"双碳"目标实现
- 提升产品环保价值
- 促进生态农业发展

## 🔧 技术架构总览

### 前端技术栈（Android）
| 技术类别 | 使用技术 | 应用场景 |
|---------|---------|---------|
| **开发语言** | Java/Kotlin | 业务逻辑开发 |
| **UI框架** | Material Design | 界面设计 |
| **异步处理** | Thread + Handler | 后台任务 |
| **网络请求** | HttpURLConnection | API通信 |
| **图像处理** | Bitmap + BitmapFactory | 图片处理 |
| **数据存储** | SharedPreferences/SQLite | 本地数据持久化 |
| **图表组件** | 自定义View/MPAndroidChart | 数据可视化 |

### 后端技术栈
| 技术类别 | 使用技术 | 应用场景 |
|---------|---------|---------|
| **Web服务** | Node.js + Express | 图片上传服务 |
| **文件处理** | Multer | 文件上传中间件 |
| **跨域处理** | CORS | 前后端通信 |
| **AI推理** | PyTorch + YOLOv5 | 目标检测 |
| **进程管理** | child_process.spawn | Python脚本调用 |

### AI模型栈
| 模型类型 | 框架 | 应用功能 |
|---------|------|---------|
| **YOLOv5** | PyTorch | 桃子品种识别 |
| **DeepSeek-R1** | Ollama | 智能对话助手 |
| **预留接口** | Stable Diffusion | 文创设计生成 |

### 通信协议
| 协议类型 | 使用场景 | 端口 |
|---------|---------|------|
| **HTTP** | 图片上传/下载 | 8081 |
| **RESTful API** | 业务数据交互 | 预留 |
| **WebSocket** | 实时数据推送 | 预留 |

### 数据流向
```
Android客户端 → HTTP上传图片 → Node.js服务器
                                      ↓
                              调用YOLOv5 Python脚本
                                      ↓
                              返回识别结果JSON
                                      ↓
Android客户端 ← 解析显示 + 加工建议
```

**未来规划**：
- IoT设备接入（传感器、无人机）
- 区块链溯源体系
- 大数据分析平台
- 跨品类扩展（其他水果、农作物）

## 🚀 技术特色总结

### 1. **端云协同架构**
- Android客户端提供流畅交互体验
- Python后端处理AI计算任务
- 降低移动端性能要求

### 2. **本地化AI部署**
- YOLOv5模型本地推理，无需云端API
- DeepSeek本地部署，数据隐私安全
- 降低网络依赖，提升响应速度

### 3. **多模型融合**
- 计算机视觉（YOLOv5）+ 大语言模型（DeepSeek-R1）
- 图像识别与知识问答相结合
- 预留生成式AI接口（Stable Diffusion）

### 4. **模块化设计**
- 六大功能独立开发、低耦合
- 统一UI风格，用户体验一致
- 易于扩展和维护

### 5. **开源技术栈**
- 前后端均使用成熟开源技术
- 降低开发成本
- 社区支持活跃

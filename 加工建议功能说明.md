# 🍑 智桃识图 - 加工建议功能

## ✨ 功能概述

在桃子识别结果中新增**加工分类建议**功能，根据桃子的甜度、成熟度、果实尺寸、综合评级等指标，智能推荐最适合的加工方式。

## 📊 判断逻辑

### 加工类型分类规则

| 加工类型 | 判断条件 | 推荐指数 |
|---------|---------|---------|
| **🍑 桃干加工** | 甜度≥12.0 + 成熟度≥70% + 重量≥200g | ⭐⭐⭐⭐⭐ 强烈推荐 |
| **🍬 糖渍果脯** | 甜度≥11.0 + 成熟度≥80% + 特级果 | ⭐⭐⭐⭐⭐ 强烈推荐 |
| **🍷 桃子酒** | 甜度≥10.0 + 成熟度≥60% + 光照≥80 | ⭐⭐⭐⭐ 推荐 |
| **🥫 桃子醋** | 甜度≥10.0 + 成熟度≥60% + 光照<80 | ⭐⭐⭐⭐ 推荐 |
| **🧃 鲜榨桃汁** | 成熟度≥85% + 重量≥180g | ⭐⭐⭐⭐ 推荐 |
| **🥫 糖水罐头** | 重量≥220g + 特级果 | ⭐⭐⭐⭐ 推荐 |
| **🍯 桃子果酱** | 甜度≥13.0 + 成熟度≥75% | ⭐⭐⭐⭐⭐ 强烈推荐 |
| **❄️ 冷冻桃肉** | 成熟度≥80% | ⭐⭐⭐ 较适合 |
| **🍧 综合加工** | 其他情况 | ⭐⭐⭐ 较适合 |

## 💡 设计原理

### 1. **基于品质的差异化建议**
- **特级果**：推荐高附加值加工（果脯、罐头、果酱）
- **一级果**：推荐大众化加工（桃汁、桃酒、桃干）
- **普通果**：推荐基础加工（冷冻、综合制品）

### 2. **基于甜度的糖分利用**
- **超高甜度（≥13.0）**：果酱（无需额外加糖）
- **高甜度（≥12.0）**：桃干（糖分保存率高）
- **中甜度（≥10.0）**：桃酒/桃醋（发酵制品）

### 3. **基于成熟度的加工适配**
- **完全成熟（≥85%）**：桃汁、冷冻桃肉（汁水丰富）
- **基本成熟（70-84%）**：桃干、果脯（需要一定硬度）
- **未完全成熟**：综合加工

### 4. **基于尺寸的产品定位**
- **大果（≥220g）**：罐头（卖相好）
- **中大果（180-219g）**：桃汁、切块加工
- **小果**：果酱、综合制品

## 🔧 技术实现

### 文件修改
- **文件**：`IdentifyPeachActivity.java`
- **新增方法**：
  - `getProcessingRecommendation()`: 判断加工类型
  - `addProcessingCard()`: 显示加工建议卡片

### 调用位置
```java
private void displayResults(PeachResult result) {
    // ... 其他结果显示 ...

    // 添加加工建议
    String[] processingRecommendation = getProcessingRecommendation(result);
    addProcessingCard(processingRecommendation[0],
                     processingRecommendation[1],
                     processingRecommendation[2]);
}
```

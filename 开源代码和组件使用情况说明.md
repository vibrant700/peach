## 五、系统架构与技术栈

### 5.1 前端技术栈

**开发语言**: Java  
**UI框架**: Android原生 + Material Design  
**最低支持版本**: Android 7.0 (API Level 24)  
**目标版本**: Android 14 (API Level 36)

**核心功能模块**:
- 慧眼识桃 - AI品种识别
- 桃宝对话 - 智能农业助手
- 桃农社区 - 经验交流平台
- 桃源工坊 - 文创设计生成
- 仓库管理 - 库存管理系统
- 碳汇市场 - 碳交易模拟

### 5.2 后端技术栈

**开发语言**: JavaScript (Node.js) + Python  
**Web框架**: Express.js  
**AI引擎**: YOLOv5 + PyTorch  
**通信协议**: HTTP + RESTful API  
**文件处理**: Multer (multipart/form-data)

### 5.3 数据流向

```
Android客户端 → HTTP上传图片 → Node.js服务器
                                      ↓
                              调用YOLOv5 Python脚本
                                      ↓
                              返回识别结果JSON
                                      ↓
Android客户端 ← 解析显示 + 加工建议
```

### 6.2 MIT协议组件

**协议要求**:
- ✅ 可商用
- ✅ 可修改

## 七、第三方服务使用说明

### 7.1 DeepSeek API

**服务类型**: 商业AI服务  
**用途**: 智能对话助手  
**计费方式**: 按Token计费  
**数据隐私**: 用户对话数据仅用于AI推理，不用于模型训练

### 7.2 阿里云DashScope API

**服务类型**: 商业AI服务  
**用途**: 文生图生成  
**计费方式**: 按生成次数计费  
**数据隐私**: 生成的图片和提示词仅用于推理，不用于模型训练

# 智桃识图 - 加工建议评估指标体系

## 📊 核心评估指标

### 1️⃣ **甜度 (Sweetness)**
- **单位**: °Bx (糖度)
- **来源**: YOLOv11模型识别 → 匹配模拟数据
- **阈值分级**:
  - 超高甜度: ≥ 13.0°Bx (适合果酱)
  - 高甜度: 12.0-13.0°Bx (适合桃干、果脯)
  - 中高甜度: 10.0-12.0°Bx (适合桃酒、桃醋)
  - 中等甜度: < 10.0°Bx (综合加工)

**决策逻辑**:
```java
if (sweetness >= 13.0) → 桃子果酱
else if (sweetness >= 12.0) → 桃干加工
else if (sweetness >= 11.0) → 糖渍果脯
else if (sweetness >= 10.0) → 桃子酒/桃醋
else → 综合加工
```

---

### 2️⃣ **成熟度 (Maturity)**
- **单位**: % (着色面积百分比)
- **来源**: YOLOv11模型识别 → 匹配模拟数据
- **阈值分级**:
  - 完全成熟: ≥ 80% (适合桃汁、冷冻桃肉)
  - 基本成熟: 60-79% (适合桃干、桃酒)
  - 未完全成熟: < 60% (综合加工)

**决策逻辑**:
```java
if (maturity >= 85) → 鲜榨桃汁
else if (maturity >= 80) → 糖渍果脯、冷冻桃肉
else if (maturity >= 70) → 桃干加工
else if (maturity >= 60) → 桃子酒/桃醋
else → 综合加工
```

---

### 3️⃣ **果实重量 (Weight)**
- **单位**: g (克)
- **来源**: YOLOv11模型识别 → 匹配模拟数据
- **阈值分级**:
  - 大果: ≥ 220g (适合罐头)
  - 中大果: 180-219g (适合桃汁)
  - 大果(桃干): ≥ 200g
  - 其他: 综合加工

**决策逻辑**:
```java
if (weight >= 220 && grade.equals("特级果")) → 糖水罐头
else if (weight >= 200) → 桃干加工
else if (weight >= 180) → 鲜榨桃汁
else → 根据其他指标判断
```

---

### 4️⃣ **综合评级 (Grade)**
- **来源**: YOLOv11模型识别 → 匹配模拟数据
- **分级标准**:
  - 特级果: 置信度95%
  - 一级果: 置信度85%
  - 二级果: 置信度75%

**决策逻辑**:
```java
if (grade.equals("特级果")) {
    if (sweetness >= 11.0 && maturity >= 80) → 糖渍果脯
    else if (weight >= 220) → 糖水罐头
}
```

---

### 5️⃣ **光照评分 (Light Score)**
- **单位**: 分数 (0-100)
- **来源**: 模拟数据
- **阈值分级**:
  - 光照充足: ≥ 80 (适合桃子酒)
  - 光照良好/一般: < 80 (适合桃子醋)

**决策逻辑**:
```java
if (sweetness >= 10.0 && maturity >= 60) {
    if (lightScore >= 80) → 桃子酒
    else → 桃子醋
}
```

---

### 6️⃣ **果径 (Diameter)**
- **单位**: mm (毫米)
- **来源**: 模拟数据
- **用途**: 辅助展示，不作为主要决策指标

**强烈推荐 (90-95分)**:
- 多项核心指标完美匹配
- 加工优势明显
- 市场价值高

**推荐 (85-89分)**:
- 主要指标匹配良好
- 加工效果较好
- 实用性强

**较适合 (78-84分)**:
- 部分指标匹配
- 可作为备选方案
- 灵活性高

## 🔍 决策优先级

### 指标重要性排序

1. **甜度** (权重: ⭐⭐⭐⭐⭐)
   - 最关键指标
   - 决定产品类型（果酱、桃干、果脯）
   - 直接影响口感

2. **成熟度** (权重: ⭐⭐⭐⭐)
   - 影响加工工艺选择
   - 决定出汁率
   - 影响保质期

3. **综合评级** (权重: ⭐⭐⭐⭐)
   - 特级果优先用于高附加值产品
   - 影响产品定价
   - 关联加工质量

4. **果实重量** (权重: ⭐⭐⭐)
   - 影响产品卖相
   - 决定是否适合做罐头
   - 影响出品率

5. **光照评分** (权重: ⭐⭐)
   - 辅助决策指标
   - 仅用于桃酒/桃醋区分
   - 影响发酵品质

## 🧮 评估公式

### 综合匹配度计算

```
匹配度 = 基础分 + 指标加分

基础分: 75分

指标加分:
- 甜度完美匹配 (+10分)
- 成熟度完美匹配 (+8分)
- 特级果 (+7分)
- 大果 (+5分)
- 光照充足 (+3分)
```

**示例计算**:

**桃干加工** (北京27号: 甜度12.5, 成熟度78%, 重量235g, 特级果)
```
基础分: 75
甜度12.5≥12.0: +10
成熟度78≥70: +8
特级果: +0 (已包含在甜度判断中)
重量235≥200: +5
光照: +0 (不影响桃干判断)
────────────────────
总分: 75 + 10 + 8 + 5 = 98分
实际显示: 95分 (经验调整)
```

## 🎓 专家经验总结

### 加工适配原则

1. **高糖+大果** → 桃干 (糖分保存率高)
2. **高糖+特级+完全成熟** → 果脯 (卖相好)
3. **超高糖** → 果酱 (无需加糖)
4. **完全成熟+中大果** → 桃汁 (出汁率高)
5. **大果+特级** → 罐头 (果形完整)
6. **中糖+光照好** → 桃酒 (发酵香醇)
7. **中糖+光照一般** → 桃醋 (营养健康)
8. **完全成熟** → 冷冻 (保持新鲜)

### 指标联动效应

**甜度 × 成熟度 联动**:
- 高糖 + 高熟 = 桃干/果酱 (最佳组合)
- 高糖 + 低熟 = 不推荐 (未成熟)
- 低糖 + 高熟 = 桃汁 (利用汁水)

**评级 × 重量 联动**:
- 特级 + 大果 = 罐头/果脯 (高端产品)
- 一级 + 中果 = 桃汁/桃酒 (大众产品)
